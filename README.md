# simple-gnome-tiling
# Simple GNOME Tiling Extension

Eine lightweight Tiling Window Manager Extension für GNOME Shell (Version 45+), inspiriert von Forge und PaperWM.

![GNOME Shell](https://img.shields.io/badge/GNOME-Shell-4A90D9?style=for-the-badge&logo=gnome&logoColor=white)
![License](https://img.shields.io/badge/License-GPLv3-blue.svg?style=for-the-badge)
![GNOME Version](https://img.shields.io/badge/GNOME-45%2B%20-purple.svg?style=for-the-badge)

## ✨ Features

| Feature | Status |
|---------|--------|
| Side-by-side Tiling | ✅ Basis |
| Floating Toggle | ✅ Basis |
| Vim-style Navigation | 🚧 Coming soon |
| Multi-monitor Support | 🚧 Coming soon |
| Grid Layouts | 🚧 Coming soon |
| Animationen | ❌ Planned |

## 🎮 Tastenkürzel

| Aktion | Taste |
|--------|-------|
| Linkes Fenster tilen | `Super + H` |
| Rechtes Fenster tilen | `Super + L` |
| Oberes Fenster tilen | `Super + K` |
| Unteres Fenster tilen | `Super + J` |
| Floating Toggle | `Super + Space` |
|Fullscreen | `Super + F` |
